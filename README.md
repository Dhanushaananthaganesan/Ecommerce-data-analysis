E-commerce Data Analysis Project

Project Overview
This project demonstrates an end-to-end data analysis workflow using Excel, MySQL, and Power BI.

The dataset ecom was split into Orders and Products, imported into MySQL, analyzed using SQL queries, and visualized through an interactive Power BI dashboard.

Tools & Technologies
- Excel (Data Cleaning)
- MySQL (Database & SQL Queries)
- Power BI (Dashboard & Visualization


Dataset
- orders.csv – Order-level transaction data
- products.csv – Product details and pricing

SQL Analysis
SQL queries include:
- Joins between orders and products
- Revenue and sales analysis
- Top-selling products
- Category-wise revenue
- Aggregate metrics (SUM, AVG, COUNT)

Power BI Dashboard
Key insights:
- Total Orders, Revenue & Items Sold
- Orders by Category
- Top Products by Quantity Sold
- Revenue by City
- Quarterly Order Trends

Data pipeline
Excel → MySQL → Power BI (via ODBC)


Key Learnings
- Data normalization & relational modeling
- SQL joins and aggregations
- Connecting MySQL to Power BI using ODBC
- Building interactive dashboards
